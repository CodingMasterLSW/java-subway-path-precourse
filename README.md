# 지하철 노선도 경로 조회 미션

## 초기 설정
- [ ] 역 초기 설정을 진행한다.
- [ ] 노선 초기 설정을 진행한다.

## 경로 조회
- [ ] 최단 거리로 경로를 조회한다
- [ ] 최소 시간으로 경로를 조회한다.

## 입력
- [x] 기능 선택을 입력받는다.
  - Exception
    - [x] 1 or Q가 아닐 경우, 예외를 발생시킨다.
- [x] 세부 기능 선택을 입력받는다.
  - Exception
    - [x] 1 or 2 or B가 아닐 경우, 예외를 발생시킨다.

- [ ] 출발역을 입력받는다.
- [ ] 도착역을 입력받는다.

## 출력
- [ ] 조회 결과를 출력한다.
